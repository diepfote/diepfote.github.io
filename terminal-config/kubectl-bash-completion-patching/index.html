Extend kubectl bash completions/extend oc bash completions

e.g. add bash completions for krew plugins

Table of contents

  • What does this do?
  • Examples
  • Where to find

What does this do?

It includes krew plugins and my kubectl plugins in the root command completion. It also provides custom completions for each of my kubectl plugins.

Examples
